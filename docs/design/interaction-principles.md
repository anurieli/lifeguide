# Interaction Principles

**Status:** ✅ The design contract every surface obeys. This is how something deep stays calm. If a design violates one of these, the design is wrong.

1. **Two beats a day, not all day.** The app asks for two minutes, twice (morning + evening). It does not buzz in between. Proactivity is earned, never scheduled.
2. **One thing per screen.** Never a dashboard. Morning shows direction + today; evening shows one prompt. Depth is *opened*, never dumped.
3. **Talk OR operate — both first-class.** The Coach is the power tool; direct manual manipulation is always available. The Coach is never a gate.
4. **Earned interruption only.** The Coach speaks unprompted only when it has something specific and true to say (max ~1–2/day). No generic check-ins.
5. **Progressive disclosure.** Day one is dead simple — one capture, one question, one pillar. Pillars, the Guide, the graph, settings depth are all earned as the user goes deeper.
6. **Ambient, not anxious.** Warm paper, soft shadows, generous space, no harsh chrome. A calm room, not a triage queue. No streaks, no guilt, no gamification.

**Tone:** the Coach is a sharp older friend — direct, warm, unhurried. Never therapy-speak ("I hear you"), never productivity hype ("crush it"), never generic encouragement.

Grounded in the references research (ritual bookends out-retain streaks; proactive tether ~75% vs wait-to-be-opened ~51%). See [`../product/concept-and-soul.md`](../product/concept-and-soul.md).
