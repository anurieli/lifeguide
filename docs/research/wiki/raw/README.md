# Raw Research Material

This folder holds source material for the LifeGuide research wiki.

Raw means: useful input, not final product direction. The synthesized map lives in [`../wiki/`](../wiki/).

## Folders

| Path | Purpose |
|---|---|
| [`sources/`](sources/) | External papers, articles, reviews, and extracted notes. |
| [`internal-notes/`](internal-notes/) | Repo-native research notes moved out of the top-level research folder and preserved as source material. |
