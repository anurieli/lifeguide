# Design System

**Status:** 🟡 outline (extracted from the prototype `../../mockup/index.html`)

## Aesthetic
Calm, warm, editorial. Off-white paper, soft shadows, rounded cards, one deep accent, gold for "emerging direction / north star." The Coach is a warm-dark, intimate panel — talking to someone in a quiet room.

## Tokens (from the prototype)
```
--paper      #FAF8F2   page background (warm paper)
--paper-2    #F3EFE5   subtle fills
--card       #FFFFFF   surfaces
--ink        #1A1D24   primary text
--ink-soft   #444B58   secondary text
--ink-mute   #8A8F9C   tertiary / labels
--line       #E7E1D4   borders
--accent     #1E3A5F   deep blue (sparingly)
--gold       #B8945A   north star / emerging direction
pillars: green #4F7A4A · blue #3A5C86 · violet #5B4B7A · teal #2F6E6A
--coach      #1E232E   Coach panel (warm dark)
```
Type: system sans (`-apple-system…`). Generous scale and spacing. Radii 12–18px. Soft layered shadows.

## Components (catalog to expand)
Node card (text/quote/image) · edge + label · toolbar · inbox tray · Coach FAB + panel + message bubbles · pillar chip · rail nav item · onboarding stepper · ritual toggle · direction/focus cards · Mirror block · settings rows/toggles/segments.
