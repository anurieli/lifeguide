VoiceField — a modular voice input for any LifeGuide field

One component. It wraps any field, carries the field's metadata (question, descriptor, prompts), and runs the same five-state arc everywhere: idle → listening → (prompt mode) → analyzing → done. Click the mic on any field below to watch it. This is a thin sketch — transcription and cleanup are scripted, not real.

Panel A · Anatomy & state machine
The element alone. Step through the states, or click the mic to run the live arc.
Idle: a text field with a pulsing mic. Type or click to speak. Field metadata is just props.
Listening: the field morphs into an elongated bar with live audio waves; words stream in minimally; Prompt Mode floats contextual "say this next" chips drawn from the question + the core.
Analyzing: on stop, the raw transcript is sent to a cleanup/intent pass that fixes speech, trims filler, and shapes it to what the field is asking.
Done: the cleaned text lands in the field, marked polished, with a one-tap show raw to revert. Nothing is silently overwritten.
