# LifeGuide Research Wiki

An LLM-maintained research wiki over raw source material. It uses Karpathy's method: source data in [`../raw/`](../raw/), synthesized notes in this folder, this index as the map, and [`log.md`](log.md) as the changelog.

Source method: https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f

## Structure

| Path | Purpose |
|---|---|
| [`../raw/`](../raw/) | Raw source material and internal notes to ingest. |
| [`../raw/sources/`](../raw/sources/) | External papers/articles with extracted notes. |
| [`../raw/internal-notes/`](../raw/internal-notes/) | Repo-native research notes preserved as raw material. |
| [`CLAUDE.md`](CLAUDE.md) | Local schema and editing contract. |
| [`log.md`](log.md) | Ingestion and reorganization log. |
